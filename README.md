# PepCompare - Peptide Price Comparison Tool

A modern web application for comparing peptide prices, delivery times, and stock levels across UK suppliers.

## Features

- Real-time price comparison across 15+ UK suppliers
- Advanced filtering by price, purity, delivery time, stock
- Live data updates via automated web scraping
- Price history tracking and trends
- Responsive design for all devices

## Tech Stack

- Next.js 14 with TypeScript
- React 18
- PostgreSQL database
- Tailwind CSS
- Apify for web scraping

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deployment

Deploy to Vercel with one click - GitHub integration ready!

## Database

PostgreSQL on Render or Railway with optimized schema for fast queries.
